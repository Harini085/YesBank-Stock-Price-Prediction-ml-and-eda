# 📈 Yes Bank Stock Price Prediction

## 📌 Project Overview
This project focuses on analyzing historical stock price data of Yes Bank and building machine learning models to predict future closing prices. The project includes comprehensive Exploratory Data Analysis (EDA), feature engineering, model development, evaluation, and performance comparison.

---

## 🎯 Objective
- Analyze historical stock price trends.
- Identify patterns and relationships among stock features.
- Engineer meaningful features to improve prediction.
- Build and evaluate regression models for stock price prediction.
- Compare model performance using standard evaluation metrics.

---

## 📊 Dataset
The dataset contains historical Yes Bank stock price information, including:

- Date
- Open Price
- High Price
- Low Price
- Close Price

---

## 🔍 Exploratory Data Analysis (EDA)
The EDA includes:

- Data cleaning
- Missing value analysis
- Descriptive statistics
- Distribution analysis
- Correlation analysis
- Time series visualization
- Outlier detection
- Feature engineering

---

## 🤖 Machine Learning Models
The following regression models were implemented:

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

---

## 📈 Evaluation Metrics
Models were evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
